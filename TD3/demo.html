Introduction

Le langage HTML (HyperText Markup Language) permet de structurer une page web, tandis que CSS (Cascading Style Sheets) gère sa mise en forme.

Dans cet article, nous allons découvrir les bases de HTML et CSS, ainsi que la manière de les relier pour créer des pages web modernes.

Les balises HTML

Les balises HTML décrivent la structure de la page. Par exemple, la balise <p> est utilisée pour définir un paragraphe.

Conseils : Utilisez des balises sémantiques (<header>, <main>, <footer>, etc.) pour donner du sens à votre document.

Sélecteurs CSS

Les sélecteurs CSS permettent de cibler précisément des éléments pour les styliser. En voici quelques exemples :

  • Sélecteurs de balise (ex: p)
  • Sélecteur d’ID (ex: #header-site)
  • Sélecteur de classe (ex: .important)
  • Sélecteurs descendants (ex: article > p)
  • Sélecteurs multiples (ex: h2, h3)
  • Pseudo-classes (ex: a:hover)
  • Pseudo-éléments (ex: p::first-line)
